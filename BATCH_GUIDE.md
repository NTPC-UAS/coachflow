# 批次課表整理

使用 [batch.html](C:\Users\User\Desktop\codex\batch.html) 可以一次整理整批課表圖片或 PDF。

## 用法

1. 打開 [batch.html](C:\Users\User\Desktop\codex\batch.html)
2. 點 `選擇課表檔案`
3. 一次多選整批 `JPG / PNG / PDF`
4. 點 `開始批次整理`
5. 左側選某一張課表，右側會顯示：
- 修正後文字
- OCR 前處理預覽
- 解析出的動作、組數、次數
6. 如有需要，可直接修改修正後文字
7. 點 `匯出 JSON`

## 適合做什麼

- 一次掃過整個資料夾
- 快速找出哪些課表辨識最差
- 把修正後的文字和解析結果整批留存

## 備註

- 這個頁面不會直接寫入 Google Sheets
- 它是整理和校正課表用的工作台
- 等你確認這批結果穩了，再把規則補回主系統
